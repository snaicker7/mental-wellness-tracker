# Generate safe coping guidance

Type: AFK
Blocked by: 1. Secure daily journal and mood capture; 2. Detect hidden stress triggers
User stories covered: Students can receive contextual coping strategies, mindfulness exercises, and motivational encouragement from a safe companion role.

## What to build

Add the conversational support path that turns current context and analysis into practical, empathetic guidance. The slice should produce short, relevant coping suggestions that feel supportive without overstepping into unsafe advice.

## Acceptance criteria

- [x] The app returns coping strategies tailored to the student's current mood and stress context.
- [x] The response set includes breathing, mindfulness, and motivational prompts.
- [x] The companion tone is empathetic, non-judgmental, and safe.
- [x] Guardrails prevent harmful, manipulative, or medically inappropriate responses.
- [x] Tests cover context-sensitive generation and safety filtering.
- [x] Coping guidance and companionship elements are integrated cleanly into the user interface (e.g. as part of the onboarding/wizard flows, "Yes or No Quiz" cards, or as context cards on the dashboard) in the visual style of [image.png](file:///c:/Users/Surya/Documents/mental-wellness-tracker/image.png).
